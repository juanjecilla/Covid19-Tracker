# Covid-19 Tracker #

Android app that shows the current status of the global pandemic of Covid-19. Data extracted from [covidAPI](https://github.com/javieraviles/covidAPI).

## Development
- Fully developed in Kotlin.
- Clean Architecture
- SOLID principles.
- AndroidX
- Libraries:
	+ Room
	+ Retrofit
	+ Koin
	+ RxJava
	+ LeakCanary
	+ BottomNavigationViewEx
	+ Mock
